快速开始
===================

以下代码告诉你如何快速开始ctpbee ::

    from ctpbee import CtpBee
    app = CtpBee("ctpbee", __name__)
    info = {
        "CONNECT_INFO": {
            "userid": "",
            "password": "",
            "brokerid": "",
            "md_address": "",
            "td_address": "",
            "appid": "",
            "auth_code": "",
        },
        "TD_FUNC": True,
    }
    app.config.from_mapping(info)  #
    app.start()



代码解释：


app = CtpBee("ctpbee", __name__) # 创建核心账户对象

app.config.from_mapping(info)  #从dict中载入信息,对于更多载入方式,请参阅 :ref:`配置文件`.

app.start() #  开启处理功能
