bee bee .... there is an industrious bee created ~~

## Download 


```
git clone https://github.com/somewheve/ctp-market
```

## Libraries

- [vnpy](https://github.com/vnpy/vnpy)

- pymongo

## Install 
```bash
python3  setup.py install
```

## quick start 
```python

from ctpbee import CtpBee
app = CtpBee(__name__)
app.start()

```
